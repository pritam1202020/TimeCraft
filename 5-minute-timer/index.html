FAQ — 5 Minute Timer

How do I start a 5-minute timer?The timer is already set to 5:00 when you open the page. Tap the green Start button, tap the ring, or press Space on a keyboard. No typing needed.
Does it make a sound when done?Yes. An audio alarm plays automatically at zero. Allow browser notifications to get an alert even if this tab is minimised.
How many seconds is 5 minutes?Exactly 300 seconds. The ring depletes by one tick every second across all 300 counts.
Can I set a different duration?Yes — use the preset chips (1, 2, 3, 5, 10, 15 min) or type any custom number in the Custom field and tap Set.
Does it keep running in the background?Yes. The countdown continues accurately as long as the tab is open. The alarm will fire correctly because we track wall-clock time, not just frame ticks.
Is this different from a Pomodoro timer?A Pomodoro timer automatically cycles through focus (25 min) and break (5 min) rounds. This is a simple single-purpose countdown — set it and it rings once.
